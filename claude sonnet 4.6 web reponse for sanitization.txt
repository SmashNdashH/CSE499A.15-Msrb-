## Issues to Fix Before Using

### 🔴 1. No `random.seed()` — Breaks Reproducibility

```python

# Fix — add this near the top of sanitize_and_augment()
random.seed(42)
```

Your faculty explicitly emphasized reproducibility. Without a fixed seed, two runs of this script produce different datasets, which means your training run can never be exactly reproduced. This matters for your IEEE report.

---

### 🔴 2. No Output Shuffle — Biased Training Batches

The output file will have all Guatemala volcano samples first, then all Hurricane Michael samples. During training, the model sees one disaster type for entire epochs before switching. Add one line before writing:

```python
random.shuffle(clean_data)  # Add before the write loop
```

---

### 🟡 3. Placeholder Check is Too Narrow

`[insert` catches the specific failure in your line 4, but the Gemini API produces other placeholder patterns too. A regex catches all of them:

```python
import re

# Replace the current check with this
PLACEHOLDER_PATTERN = re.compile(r'\[.*?\]')

if PLACEHOLDER_PATTERN.search(row['response']):
    print(f"🧹 Removed Dirty Sample (Placeholder): {row['image']}")
    dirty_count += 1
    continue
```

This also catches `[coordinates]`, `[location not specified]`, `[TBD]`, etc.

---

### 🟡 4. No Missing Key Protection — Will Crash on Malformed JSON

If any line in your JSONL is missing the `response` or `image` key (easy to happen if the Gemini API returned an incomplete object), `row['response']` throws a `KeyError` and crashes the entire script, losing all progress. Wrap it:

```python
row = json.loads(line)

# Add this guard
if not all(k in row for k in ('image', 'instruction', 'response')):
    print(f"⚠️  Skipped malformed entry: {line.strip()[:80]}")
    dirty_count += 1
    continue
```

---

### 🟢 5. Instruction Distribution is Uneven by Chance

With `random.choice` across 6 variants on ~220 samples, you could end up with Variant 1 appearing 60 times and Variant 5 appearing 20 times purely by chance. A round-robin guarantees balance:

```python
# Replace random.choice with this
row['instruction'] = INSTRUCTION_VARIANTS[len(clean_data) % len(INSTRUCTION_VARIANTS)]
```

## The Fixed Script (Drop-in Replacement)

```python
import os, json, re, random
from collections import Counter

PROJECT_ROOT = os.path.dirname(os.path.dirname(os.path.abspath(__file__)))
INPUT_JSONL  = os.path.join(PROJECT_ROOT, "dataset", "verified_dataset.jsonl")
OUTPUT_JSONL = os.path.join(PROJECT_ROOT, "dataset", "final_training_dataset.jsonl")

INSTRUCTION_VARIANTS = [
    "Analyze this aerial view and identify priority zones for search and rescue operations.",
    "What are the key rescue priorities visible in this post-disaster aerial image?",
    "Assess this disaster scene for survivor localization and structural hazards.",
    "Identify structural failures and safe extraction routes from this satellite image.",
    "Provide a tactical rescue operations briefing based on this aerial view.",
    "Evaluate the structural damage and highlight logistics constraints in this area."
]

PLACEHOLDER_PATTERN = re.compile(r'\[.*?\]')

def sanitize_and_augment():
    random.seed(42)  # Reproducibility

    if not os.path.exists(INPUT_JSONL):
        print(f"ERROR: {INPUT_JSONL} not found. Run QA Validator first.")
        return

    clean_data = []
    disaster_counts = Counter()
    dirty_count = 0

    with open(INPUT_JSONL, 'r', encoding='utf-8') as f:
        for line_num, line in enumerate(f, 1):
            if not line.strip():
                continue

            try:
                row = json.loads(line)
            except json.JSONDecodeError:
                print(f"⚠️  Line {line_num}: Invalid JSON — skipped.")
                dirty_count += 1
                continue

            # Guard: missing keys
            if not all(k in row for k in ('image', 'instruction', 'response')):
                print(f"⚠️  Line {line_num}: Missing keys — skipped.")
                dirty_count += 1
                continue

            # Guard: placeholder text
            if PLACEHOLDER_PATTERN.search(row['response']):
                print(f"🧹 Line {line_num}: Placeholder found — removed: {row['image']}")
                dirty_count += 1
                continue

            # Instruction diversity (round-robin for balance)
            row['instruction'] = INSTRUCTION_VARIANTS[len(clean_data) % len(INSTRUCTION_VARIANTS)]

            # Disaster type tracking
            filename = os.path.basename(row['image'])
            disaster_counts[filename.split('_')[0]] += 1

            clean_data.append(row)

    # Shuffle before writing (prevents batch bias)
    random.shuffle(clean_data)

    with open(OUTPUT_JSONL, 'w', encoding='utf-8') as f_out:
        for row in clean_data:
            f_out.write(json.dumps(row) + '\n')

    print("\n" + "="*50)
    print("📊 DATASET SANITIZATION & DISTRIBUTION REPORT")
    print("="*50)
    print(f"Total Original Samples : {len(clean_data) + dirty_count}")
    print(f"Dirty Samples Removed  : {dirty_count}")
    print(f"Final Clean Samples    : {len(clean_data)}\n")
    print("🌪️  Disaster Type Distribution:")
    for disaster, count in disaster_counts.most_common():
        print(f"  - {disaster.ljust(25)}: {count} ({count/len(clean_data)*100:.1f}%)")
    print("="*50)
    print(f"✅ OUTPUT: {OUTPUT_JSONL}")

if __name__ == "__main__":
    sanitize_and_augment()
```
